# Elena's Portfolio

A modern, responsive portfolio website showcasing Elena's passion for taekwondo, mathematics, economics, and creative projects.

## 🚀 Features

- **Modern Design**: Clean, professional, and visually appealing layout
- **Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Sections**:
  - Hero Section with engaging introduction
  - About Me - Personal story and background
  - Skills & Strengths - Taekwondo, Math, Economics, and Critical Thinking
  - Projects - Magic Box and other activities
  - Navigation bar for easy access

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - Modern styling with animations and responsive design
- `README.md` - This file

## 🎨 Design Highlights

- Blue gradient hero section
- Smooth animations and transitions
- Hover effects on cards and buttons
- Mobile-friendly responsive layout
- Professional color scheme

## 🌐 Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize with your own content, images, and links
